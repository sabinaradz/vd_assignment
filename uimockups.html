UI Mockups

These are the mockups I created for my web project. I utilised the grid method by 960.gs to be able to create a visually pleasing, proportionate and neat layout.

Homepage

For this website, I chose to utilise a simplified navigation bar with a universally recognised 'hamburger' icon representing the menu. the presence of simple but informative icons to replace text allows for greater accessibility when interacting with users from different regions, and the absence of words makes the bar appear more streamlined. of the world. I chose to include a search bar on the navigation bar so that it is simple for the user to find what they are looking for. The footer includes information that isn't necessarily what a user looks for when shopping but that is important to include. As is standard I have included these at the bottom of the page where most modern websites store these links.

Menu Extended

I chose to create a mockup of the 'hambuger' menu extended to see how it would complement the layout of the website. I chose to maintain the flow of the banner by keeping the background colour the same and centering the elements to make theme easy to see instantly upon opening the menu.

Product Page

For the product page, I chose a typical layout of 4 images per row and included filter/sort options for the user to be able to narrow down the selection of products and made both of these highly visible at the top. In future iterations, I could also add an option to show less or more tiles per row for users that need easier visibility of each item.

Help Page

The final mockup I have created is for a help page, that has an embedded YouTube tutorial video. In keeping with the minimalistic theme of the website, I chose to use a visual aid instead of a large amount of text.
